# grow-graphql-api

[Grow-IoT](https://github.com/CommonGarden/Grow-IoT) graphql api

## Installation

```sh

yarn
#npm install
yarn run compile

```

start Grow-IoT app

```sh
#export MONGO_URL=mongodb://localhost:27017/db_name

#npm start

```
